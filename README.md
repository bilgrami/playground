# playground

A workspace for data tooling, ingestion utilities, and reference implementations.

## Summary
- `snowflake-diff`: Snowflake drift detection and diff reporting (SnowCLI-based).
- `mongodb`: JSON flattening toolkit with CSV output and Docker-based MongoDB ingest.
- `2024-01-csvImportToMaster`: CSV merge/import helper scripts.
